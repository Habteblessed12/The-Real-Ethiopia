# The Real Ethiopia

A visually engaging website showcasing the beauty and culture of Ethiopia using HTML and CSS.  

---

## 🌍 Features

- **Colorful Header:** Represents Ethiopia’s national colors (Green, Yellow, Red)  
- **Welcome Section:** Introduces visitors to Ethiopia’s landscapes and culture  
- **YouTube Section:** Embed videos to share Ethiopian travel or culture content  
- **Photo Cards:** Highlight popular destinations: Awash, Wenchi, Arbaminch, and Hawassa  
- **Responsive Design:** Works on desktop and mobile devices  

---

## 🖥️ Technology

- HTML  
- CSS
